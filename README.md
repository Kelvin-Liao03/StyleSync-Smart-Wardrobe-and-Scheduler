# StyleSync: Smart Wardrobe & Scheduler

A smart wardrobe management app designed to help users organize clothing, plan outfits, and receive personalized recommendations according to weather conditions.

## Features

Add, edit, and delete clothing items

Track outfit history and manage event-based outfits

Receive low inventory alerts for clothing items

Prevent re-selection of worn outfits until restored

User profile with login and registration

Local notifications for event and outfit reminders

Color and style matching for outfit suggestions

Calendar-based outfit scheduling

## Technologies

Programming Language: C#

Framework: .NET MAUI

Database: SQLite

UI Design: XAML

IDE: Visual Studio

Architecture: MVVM (Model-View-ViewModel) pattern

Notifications: Local notifications API

Data Management: DatabaseService with CRUD operations

Platform: Cross-platform (Android, iOS, Windows)
